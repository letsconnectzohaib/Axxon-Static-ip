# Static IP Configuration Script

## Purpose
Converts dynamic IP to static IP with automatic detection and user data storage.

## How to Use

- Right-click `Set-StaticIP-Standard.bat`
- Select "Run as administrator"
- Script automatically detects current IP settings
- Press Enter to confirm configuration
- Enter your name when prompted
- Script saves data to Google Sheet

## Requirements

- Windows Administrator privileges
- Internet connection for Google Sheets
- Network adapter named "Ethernet"

## Features

- Automatic IP detection
- Static IP configuration
- Google DNS (8.8.8.8, 8.8.4.4)
- User data storage
- Google Sheets integration
- Local file backup

## Files

- `Set-StaticIP-Standard.bat` - Main script
- `StaticIP_Database.txt` - Local backup file
